# HTTP-Server-System
For this project, I created HTTP server that serves Pre-Departure information as a JSON response within Python.
The file contains the following information:
- Aircraft Status: Ready for departure
- Passenger Boarding Number: 128
- Fueling: Completed
- Door State: Closed
- Push Back Time: 13:45

Here is the scenario I used:

Aircraft activities

You are working for Aero Airlines, an airline operating in South Africa and linking different cities. You are in charges of monitoring the airplane system focusing on Air traffic control and airline operations center using ADS-B, radar, and satellite systems.

The ground tracked activities include:

Tracked Activities on the Ground:

1.     Pre-Departure (at the Gate):

o   Aircraft status: Ground crew updates the maintenance log via electronic systems.

o   Passenger boarding: Count and verification logged into the airline’s database.

o   Fueling and catering: Automated IoT systems on refueling trucks log the amount delivered.

o   Door closure and pushback time: Timestamped and tracked in real-time.

2.     In-Flight Events:

o   Autopilot status: Logged internally by the airline’s onboard flight data monitoring (FDM).

o   Cabin pressure: Monitored and transmitted periodically to ground-based maintenance platforms.

o   Wi-Fi usage: Logged for service quality tracking.
